HTML

  • HTML stands for Hyper Text Markup Language and is the most basic building block of websites. It handles the content, structure, and meaning of each element on a page.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element defines the title of the webpage.
  • The link element connects an external image or css resource to the HTML document.
  • The script element embeds client-side scripts in the HTML document.

CSS

  • CSS stands for Cascading Style Sheets which is the language used to describe the presentation of a website created with HTML and how each element on a page is rendered.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The id selector matches an element based on the value of the element's id attribute. In order for the element to be selected, its id attribute must match exactly the value given in the selector.
  • Declarations contain properties we want to apply and the value of the property.
  • Properties include font-size, text-align, (text)color, and background-color.

Git

  • Git is a free open source version control system that allows web developers to collaborate on projects without writing over each others' work. It also allows developers to revert to previous versions of code if a problem is discovered.
  • The git command must precede every git statement.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • The checkout command is used to move the working branch to a new one.
  • The -b flag creates a new branch.
  • The add command adds modifications in the current working branch to the staging area
  • The-A Flag indicates that we want to add all changes
  • The commit command is used to save your changes to the local repository.
  • The -m flag indicates the message contained in quotes that describes the commit command.

JavaScript

  • JavaScript is a lightweight programming language used to provide logic and interactivity for websites.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • The primitive data types used in JavaScript are number, string, boolean, null, and undefined.
  • Strings represent text and must be wrapped in quotes. "" and '' are both fine, but make sure to stay consistent with which quote you use!
